# 🤖 Copiloto IA Prompts

![IA](https://img.shields.io/badge/IA-Assistente%20Inteligente-blue)
![Prompt](https://img.shields.io/badge/Prompt-engineering-yellow)

Este repositório contém exemplos de **prompts utilizados com ferramentas de Inteligência Artificial**, como **GitHub Copilot e ChatGPT**.

O objetivo é demonstrar como estruturar prompts para orientar a IA a gerar respostas mais claras, úteis e organizadas durante tarefas de desenvolvimento e estudo.

---

# 📌 Objetivo do Projeto

Este projeto foi criado para:

* Aprender conceitos de **engenharia de prompts**
* Melhorar a interação com ferramentas de **Inteligência Artificial**
* Organizar prompts reutilizáveis
* Demonstrar exemplos de prompts aplicados a **desenvolvimento e estudo**

---

# 📂 Estrutura do Repositório

Os prompts estão organizados nos seguintes arquivos:

```
prompt-agent.md
prompt-ask.md
prompt-plan.md
prompt-study.md
```

Cada arquivo contém um **prompt com uma finalidade específica**.

---

# 📄 Descrição dos Arquivos

## prompt-agent.md

Este arquivo contém um prompt que orienta a IA a agir como um **agente especializado**, auxiliando na resolução de problemas ou tarefas relacionadas à programação.

Esse tipo de prompt ajuda a IA a assumir um **papel específico**, tornando as respostas mais direcionadas e contextualizadas.

---

## prompt-ask.md

Este prompt é utilizado para **fazer perguntas estruturadas para a IA**.

Ele ajuda a obter respostas mais completas, organizadas e detalhadas sobre determinado tema ou problema.

Pode ser usado para:

* Tirar dúvidas
* Entender conceitos
* Explorar soluções para problemas técnicos

---

## prompt-plan.md

Este prompt orienta a IA a **criar planos ou estratégias** para resolver uma tarefa.

Pode ser utilizado para:

* Planejar um projeto
* Definir etapas de desenvolvimento
* Estruturar soluções para um problema

---

## prompt-study.md

Este prompt é voltado para **auxiliar nos estudos**.

Ele orienta a IA a explicar conteúdos de forma didática, facilitando o aprendizado de novos conceitos.

Pode ser utilizado para:

* Explicações de conceitos técnicos
* Resumos de conteúdos
* Apoio no aprendizado de programação

---

# 🚀 Como Utilizar

1. Escolha um dos arquivos de prompt do repositório
2. Copie o conteúdo do prompt
3. Cole em uma ferramenta de IA (Copilot ou ChatGPT)
4. Adapte conforme sua necessidade
